    Jackblog 是使用 Node.js + MongoDB + 其它客户端框架, 开发的个人博客系统,前后端分离,仿简书模板.

    https://github.com/jackhutu/jackblog-vue
    此为Vue版客户端,需要配合API服务端运行.
    主要使用的库有 Vue, Vuex, Vue-resource, Vue-router, Vue-validator 等. 响应式布局.
    Demo地址: http://vue.jackhu.top
    服务端参考:
    https://github.com/jackhutu/jackblog-api-express
    或者
    https://github.com/jackhutu/jackblog-api-koa
